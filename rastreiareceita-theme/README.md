# Rastreia Receita Theme 👤

A blue-first theme crafted for Rastreia Receita and modern development workflows.

## About Rastreia Receita

<a href="https://rastreiareceita.com"><b>Rastreia Receita</b></a> is a revenue attribution analytics platform built for Brazilian entrepreneurs who sell online. It connects payment processors (Stripe, Hotmart, Kiwify, AbacatePay, Cart Panda, and more) to a real-time dashboard that shows exactly which marketing channel generates each R$ of revenue — not just pageviews. With a 4kb ultra-fast script, 60-second setup, Revenue per Visitor as its core metric, full click-to-purchase customer journey tracking, Google Search Console integration, and a 100% Portuguese interface with values in R$. Built for infoprodutores, SaaS founders, e-commerces, and agencies who want to stop guessing and start scaling what actually makes money.
