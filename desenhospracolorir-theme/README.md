# Desenhos pra Colorir Theme 👤

A blue-first theme crafted for Desenhos pra Colorir and modern development workflows.

## About Desenhos pra Colorir

<a href="https://desenhospracolorir.com"><b>Desenhos pra Colorir</b></a> is an AI-powered coloring page generator that creates custom drawings ready to print in just 4 seconds. Type a theme in Portuguese, upload a photo, or send a sketch — the AI produces clean black-and-white line art optimized for any printer. With 15+ specialized styles (cartoon, mandala, manga, zentangle and more), batch generation of up to 50 pages at once, consistent character creation for book series, and Word Art for personalized lettering — all fully in Portuguese with payment in BRL via PIX. Used by 8,400+ Brazilian families and educators to replace hours of Google searching with instant, unique coloring pages. The first Brazilian platform dedicated to AI-generated coloring content.
